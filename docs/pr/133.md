## PR description (copy/paste into the PR body)

### Motivation
The price fetcher previously relied on per-symbol quote calls, which increases latency and makes it easier to hit rate limits. This change switches to Polygon’s snapshot quote batch endpoint so we can fetch multiple quotes in a single request, improving performance and reducing external API chatter.

### What changed
- Added a batch snapshot quote fetcher to the Polygon provider (`fetchPolygonSnapshotQuotes`) with a dedicated `PolygonSnapshotQuote` type.
- Refactored the price fetcher to use Polygon batch snapshots for asset prices and extended quote data instead of making per-symbol requests.
- Updated the Market Notifications panel header to show service logos for Polygon, Finnhub, and Grok.
